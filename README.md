# Davivet — Clínica Veterinaria
## Estructura del proyecto

```
davivet/
├── index.html          ← Página principal (abrir en navegador)
├── README.md           ← Este archivo
└── img/                ← Carpeta de imágenes (poner aquí tus fotos)
    ├── hero-perros.jpg           → Hero: foto de perros/gatos (460x460px)
    ├── about-perro-gato.jpg      → Sobre Nosotros: perro y gato juntos (600x450px)
    ├── quirofano.jpg             → Instalaciones: foto del quirófano (400x320px)
    ├── ecografia.jpg             → Instalaciones: sala de ecografía (400x320px)
    ├── hospitalizacion.jpg       → Instalaciones: sala de hospitalización (400x320px)
    ├── sala-visitas.jpg          → Instalaciones: sala de visitas (400x320px)
    ├── dr-david-fonseca.jpg      → Equipo: foto Dr. David Fonseca (400x300px)
    ├── dra-lorena-galvis.jpg     → Equipo: foto Dra. Lorena Galvis (400x300px)
    ├── avatar-carolina.jpg       → Testimonio: avatar Carolina (44x44px o más)
    ├── avatar-andres.jpg         → Testimonio: avatar Andrés (44x44px o más)
    ├── avatar-mariajose.jpg      → Testimonio: avatar María José (44x44px o más)
    ├── blog-ecografia.jpg        → Blog: imagen artículo ecografía (400x200px)
    ├── blog-cirugia.jpg          → Blog: imagen artículo cirugía (400x200px)
    └── blog-emergencias.jpg      → Blog: imagen artículo emergencias (400x200px)
```

## Instrucciones

1. Pon tus fotos reales en la carpeta `img/` con los nombres exactos de arriba
2. Las medidas entre paréntesis son recomendadas (mínimo), pueden ser más grandes
3. Formato: .jpg o .png (si usas .png, cambia la extensión en el index.html)
4. Abre `index.html` en tu navegador para ver la página
